# Autonomia — REPORT CC

O usuário autoriza execução autônoma de **todas as ações** neste projeto sem pedir confirmação prévia:

- Criar arquivos, pastas, relatórios
- Editar qualquer arquivo de skill, memória, referência
- Sincronizar cópias de orgs.json, queries.md e demais arquivos
- Renomear pastas e arquivos
- Executar queries ODBC, gerar deck.html, renderizar PDF
- Atualizar memórias e documentação

**Única exceção:** perguntar antes de **excluir** arquivos ou pastas.

**Why:** Pedido explícito do usuário em 27/06/2026 para eliminar interrupções desnecessárias no fluxo de geração de relatórios.

**How to apply:** Executar diretamente, sem perguntar "posso fazer X?". Apenas informar o que foi feito.
